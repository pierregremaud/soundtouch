# soundtouch
Python script that monitors a soundtouch device and streams live radio when a preset key is pressed

Due to Bose recent and scandalous "Bose End-of-Life Announcement for SoundTouch Products" I created this script.

Install it as a service on a raspberry pi.

It will monitor your bose soundtouch device and stream live radio when a preset is selected.

I was not able to do the same with spotify playlist. I also failed to modify presets.
